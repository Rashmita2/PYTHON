Classes has the data which is represented by fields and behavior which is represented by the methods or functions.  
Object is an instance of a class.  
Static method is accessible from class, not from objects.  
Constructor has the same name as a class. It initializes an object.  
Object initializer is needed to avoid creating multiple constructors.  
Overloading methods - Same name but different parameters.
Overriding methods - Same name and same parameters.
